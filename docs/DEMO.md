# PitchCube 产品演示指南

## 启动方式

### 方式一：一键启动（推荐）
双击运行 `launch_demo.bat`

这将自动：
1. 安装依赖
2. 启动后端 (http://localhost:8000)
3. 启动前端 (http://localhost:3000)
4. 打开浏览器

## API 端点一览

| 功能 | 端点 | 方法 |
|------|------|------|
| 健康检查 | `/api/v1/health` | GET |
| 获取模板 | `/api/v1/posters/templates` | GET |
| 生成海报 | `/api/v1/posters/generate` | POST |
| 查询状态 | `/api/v1/posters/generations/{id}` | GET |
| 下载图片 | `/download/{filename}` | GET |

完整API文档: http://localhost:8000/docs

## 演示要点（给评委）

### 1分钟版本
1. "这是我们的PitchCube，AI路演展示自动化平台"
2. 打开首页 → 点击"立即体验"
3. 选择"海报工坊" → 点击"示例1"一键填充
4. 点击"开始生成"
5. 3秒后: "看，海报已经生成了，这是真实的图片文件"
6. 点击下载展示

### 技术亮点
- ⚡ **无需AI API**: 使用PIL直接渲染，零成本
- 🚀 **极速生成**: 本地渲染，3-5秒出图
- 🎨 **精美模板**: 4种风格，专业设计
- 📱 **即开即用**: 一键启动，无需配置
