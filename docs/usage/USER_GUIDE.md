## 快速开始

### 第一步：准备产品信息

在开始之前，请准备好以下产品信息：

- **产品名称**：简洁明了的产品名称
- **产品描述**：1-2句话描述产品是什么，解决什么问题
- **核心功能**：3-5个关键功能点
- **目标受众**：产品面向的用户群体

### 第二步：生成海报

1. 访问 [http://localhost:3000/generate](http://localhost:3000/generate)
2. 选择 **"海报工坊"**
3. 输入产品信息
4. 选择模板风格
5. 点击 **"开始生成"**
6. 等待 10-30 秒
7. 预览并下载海报

### 第三步：生成视频脚本

1. 进入 **"视频演播室"**
2. 选择已有的产品
3. 设置视频时长（60-180秒）
4. 选择脚本风格
5. 生成并预览视频脚本
6. 可导出为多种格式

## 功能详解

### 🎨 海报工坊

#### 模板选择

| 模板名称 | 适用场景 | 特点 |
|----------|----------|------|
| 科技现代 | SaaS、AI产品 | 蓝色渐变，简洁大气 |
| 创业宣言 | 初创路演 | 鲜明配色，突出亮点 |
| 极简主义 | 设计工具 | 黑白配色，专注内容 |
| 创意渐变 | 创意产品 | 多彩背景，展现活力 |

#### 自定义选项

- **品牌颜色**：上传品牌色值（HEX格式）
- **Logo上传**：支持 PNG、SVG 格式
- **字体选择**：系统提供 5 种字体风格
- **尺寸规格**：
  - 微信文章封面 (900×383)
  - Twitter/微博 (1200×675)
  - LinkedIn (1200×627)
  - 印刷海报 (A4, A3)

### 🎬 视频演播室

#### 脚本结构

标准路演视频脚本包含以下场景：

1. **开场** (15秒)
   - 引起注意
   - 提出问题

2. **产品介绍** (30秒)
   - 解决方案
   - 核心功能演示

3. **价值主张** (30秒)
   - 独特优势
   - 用户收益

4. **结尾** (15秒)
   - 行动号召
   - 联系方式

#### 风格选项

- **专业正式**：适合B2B产品、投资人路演
- **轻松活泼**：适合C端产品、社交媒体
- **技术深入**：适合开发者大会、技术分享

## 最佳实践

### 产品描述撰写指南

#### 好的描述示例 ✅

```
产品名称：PitchCube
产品描述：AI驱动的路演展示自动化平台，帮助黑客松团队和初创公司在10秒内生成专业路演物料，节省90%的设计时间。
核心功能：
- 智能海报生成
- 视频脚本创作
- 3D IP形象设计
- 多平台格式适配
```

#### 避免的问题 ❌

```
❌ 过于简短："一个好用的工具"
❌ 过于冗长：技术细节过多
❌ 模糊不清：没有突出核心卖点
❌ 拼写错误：影响专业形象
```

### 海报设计建议

1. **配色原则**
   - 主色不超过3种
   - 使用品牌色增强识别度
   - 确保文字对比度足够

2. **文案技巧**
   - 标题不超过10个字
   - 使用数字增强说服力
   - 突出核心价值主张

3. **视觉层次**
   - 标题最醒目
   - 产品名称次之
   - 详细信息最小

### 视频脚本优化

1. **开场黄金15秒**
   - 直击痛点
   - 引发共鸣
   - 建立期待

2. **核心展示**
   - 展示而非讲述
   - 使用具体数据
   - 对比突出优势

3. **结尾行动**
   - 明确的CTA
   - 简单的下一步
   - 留下深刻印象

## 常见问题

### Q: 生成的海报可以商用吗？

**A:** 可以。使用 PitchCube 生成的海报归你所有，可以自由商用。但请注意：
- 上传的 Logo 和品牌素材需确保你有使用权
- 某些模板字体可能需要商业授权

### Q: 没有 API 密钥可以使用吗？

**A:** 可以！没有 API 密钥时，系统会使用预设模板和智能填充功能生成海报。API 密钥用于增强 AI 生成效果，但不是必需的。

### Q: 生成的文件可以编辑吗？

**A:** 
- **海报**：可以下载 PNG 和 PDF 格式，PDF 支持矢量编辑
- **视频脚本**：可以导出为 Word、Markdown 格式进行编辑
- **IP模型**：导出 STL/OBJ 后可用 3D 软件编辑

### Q: 如何保护我的产品信息？

**A:** 
- 所有数据传输使用 HTTPS 加密
- 产品信息存储在私有数据库中
- 不会将你的数据用于训练 AI 模型
- 可随时删除账户和数据

### Q: 支持哪些语言？

**A:** 当前版本支持：
- 中文（简体）
- 英文

后续版本将支持更多语言。

### Q: 生成失败怎么办？

**A:** 
1. 检查网络连接
2. 重新提交生成请求
3. 尝试简化产品描述
4. 联系技术支持

## 高级技巧

### 批量生成

```bash
# 使用 API 批量生成海报
curl -X POST http://localhost:8000/api/v1/posters/batch \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "products": [
      {"name": "Product 1", "description": "..."},
      {"name": "Product 2", "description": "..."}
    ]
  }'
```

### 自定义模板

1. 在 `templates/custom/` 目录添加模板
2. 配置模板参数（见 API 文档）
3. 重启后端服务

### 集成 CI/CD

```yaml
# GitHub Actions 示例
- name: Generate Pitch Materials
  run: |
    curl -X POST ${{ secrets.PITCHCUBE_API }}/posters/generate \
      -H "Authorization: Bearer ${{ secrets.PITCHCUBE_TOKEN }}" \
      -d @product-info.json
```
